## 🚀 Overview
This is a terminal-based application that demonstrates core programming concepts and logic building using JavaScript.

## 🛠️ Tech Stack
- JavaScript (Node.js)
- Readline (for user input)

## ▶️ How to Run
1. Install Node.js
2. Clone the repository
3. Run the file:
   node filename.js

## 📌 Features
- User input handling
- Loop-based logic
- Conditional operations
- Real-world problem-solving scenarios
